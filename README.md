# Aether‑Guard

Aether‑Guard is an AMD‑accelerated, privacy‑first, and explainable AI cybersecurity solution designed to protect students and educational institutions from phishing attacks, malware, fake documents, and identity misuse.

---

## Problem Statement

Students and educational institutions are increasingly targeted by phishing emails, fake internship scams, malware‑infected documents, and identity misuse. Existing cybersecurity tools are complex, reactive, and designed for IT professionals, making them ineffective for student users and insufficient for building cybersecurity awareness.

---

## Solution Overview

Aether‑Guard acts as a digital security companion that provides real‑time threat detection while explaining risks in simple, human‑readable language. The system is privacy‑preserving by design and helps users learn safe digital behavior through teach‑back guidance.

---

## Key Features

- Real‑time phishing and malicious link detection  
- Malware and tampered document detection  
- Suspicious login and identity misuse detection  
- Explainable alerts in plain language  
- Teach‑back guidance for digital hygiene  
- Privacy‑preserving on‑device / on‑campus inference  
- Institutional dashboard with privacy‑safe insights  

---

## System Architecture

Users (Students / Faculty)  
→ Emails, Files, Login Requests  
→ AI Security Engine (AMD Accelerated)  
→ Risk Scoring & Explainability  
→ Privacy Layer (No Raw PII)  
→ User Alerts & Teach‑Back  
→ Institutional Dashboard

## Process Flow

1. User receives an email, uploads a file, or logs in  
2. AI engine analyzes content and behavior  
3. Risk score is generated  
4. Explainable alert is shown to the user  
5. Teach‑back guidance is provided  
6. Privacy‑safe alert summary sent to dashboard
